# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-23

### Added
- Initial repository structure
- Backend: FastAPI application skeleton with routers
- Frontend: React + TypeScript scaffolding
- Drift detection detectors: PSI, KS-Test, JS Divergence, CUSUM, Page-Hinkley, IForest
- Novel features: EWMA thresholds, CW-PSI, drift classifier, STL decomposition, calibration curves
- Docker Compose for local development
- Infrastructure templates for K8s
